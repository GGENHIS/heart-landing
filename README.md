# 💝 Heart Landing

![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-deployed-brightgreen?style=flat-square&logo=github)

Романтическая веб-страница с анимацией сердца и терминальными эффектами.

---

## ✨ Что внутри

### Экран 1 — Терминал
Тёмный фон, мерцающий текст **"Love you."** с эффектом печатающего терминала. Кнопка `DECRYPT MESSAGE` запускает переход к следующему экрану.

### Экран 2 — Сердце
Интерактивное сердце, нарисованное на `<canvas>` из слов *"i love you"* с красным свечением. Кликни по нему — и откроется финальное сообщение.

### Экран 3 — Сообщение
Личное зашифрованное послание — для того, кто знает, как его прочитать.

---

## 🚀 Как использовать

1. Открой страницу: [https://ggenhis.github.io/heart-landing/love.html](https://ggenhis.github.io/heart-landing/love.html)
2. Нажми кнопку **DECRYPT MESSAGE**
3. Кликни по сердцу ❤️

---

## 🎵 Музыка

Страница рассчитана на фоновую музыку **Massive Attack — Angel**.

Для локального использования:
- Скачай `angel.mp3`
- Положи файл рядом с `love.html`
- Открой `love.html` в браузере — музыка заиграет автоматически

> На GitHub Pages музыка не хранится (`.gitignore`), чтобы не нагружать репозиторий.

| Технология | Использование |
